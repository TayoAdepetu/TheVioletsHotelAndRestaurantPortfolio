<h2>A Little About TheViolets Hotel Project</h2>

<p>The TheViolets Hotel and Restaurant Project is a web application built for <a href="https://hotel.theviolets.ng">TheViolets Hotel</a> and <a href="https://restaurant.theviolets.ng">heViolets Restaurant</a> in Anambra State, Nigeria.</p>

<p>The software was built with Code Igniter, a PHP Framework.</p>

<p>The TheViolets Hotel and Restaurant technology has all needed features for running modern hotels and restaurants businesses technologically, including Inventory Management, Customer Management, Sales Management, Room Booking Management, Point of Sale (for payment) System, Assets Monitoring, Human Resource Management etc.</p>

<h2>Staff Role Management</h2>

<p>The TheViolets Hotel and Restaurant Web Application lets the company manage all aspects of their business, including assigning roles and permissions to all employees in such a way that an employee can only do what is permitted by their access level within the system. This ensures security and privacy of data.</p>

<h2>Hotel and Restaurant Reservation</h2>

<p>The technology lets customers make both hotel and restaurant reservations, including room reservation, restaurant table reservation, food ordering, payments, and more. The users have access to intuitive user interfaces that allow them to do all these and more.</p>

<h2>Kitchen Management</h2>

<p>The system is also used to monitor what is going on in the kitchen and store. It is designed to be reactive in such a way that all food orders and preparations start and end on the system, and all the necessary data, inclduing payments, cooking items, and store are updated in realtime as they are being used and sold.</p>

<h2>Realtime Statistics</h2>

<p>The technology lets the business manage operations happening in both the hotel and restaurant from one single dashboard, filter statistics based on each customer, staff, menu item, room type, etc.</p>
